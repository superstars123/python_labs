## Реализованный класс коллекции

**PlayerCollection**

Контейнер для хранения и управления объектами `Player`.

---

##  Описание класса

`PlayerCollection` — это контейнерный класс, который управляет группой игроков.


##  Основные возможности

| Функция    | Описание                                |
| ---------- | --------------------------------------- |
| Хранение   | Список объектов Player                  |
| Управление | Добавление, удаление, доступ по индексу |
| Поиск      | По имени, диапазону уровней, статусу    |
| Сортировка | По уровню, имени                        |
| Фильтрация | Активные, неактивные, высокий уровень   |
| Итерация   | Поддержка `for`                         |
| Защита     | Проверка типа, запрет дубликатов по id  |

---

## Методы коллекции

###  Базовые операции

| Метод            | Описание                |
| ---------------- | ----------------------- |
| add(item)        | Добавить игрока         |
| remove(item)     | Удалить игрока          |
| remove_at(index) | Удалить по индексу      |
| get_all()        | Получить список игроков |

---

### Поиск

| Метод                         | Описание         |
| ----------------------------- | ---------------- |
| find_by_name(name)            | Поиск по имени   |
| find_by_level_range(min, max) | Поиск по уровню  |
| find_by_status(status)        | Поиск по статусу |

---

### Магические методы

| Метод       | Описание           |
| ----------- | ------------------ |
| **len**     | Количество игроков |
| **iter**    | Итерация           |
| **getitem** | Индексация и срезы |
| **str**     | Вывод коллекции    |

---

### Сортировка

| Метод              | Описание                 |
| ------------------ | ------------------------ |
| sort(key, reverse) | Универсальная сортировка |
| sort_by_level()    | По уровню                |
| sort_by_name()     | По имени                 |

---

###  Фильтрация (возвращают новую коллекцию)

| Метод                     | Описание           |
| ------------------------- | ------------------ |
| get_active()              | Активные игроки    |
| get_inactive()            | Неактивные игроки  |
| get_high_level(threshold) | Игроки выше уровня |
